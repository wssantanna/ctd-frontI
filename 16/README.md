
# Aula 16 - Seletores CSS 3/3

## Pseudo-classes
Estado de um elemento.

### Estados de link(s)

```css
a:link {}
a:visited {}
a:hover {}
a:active {}
```

### Estados de input(s)

```css
input:focus {}
input:enabled {}
input:disabled {}
```

## Pseudo-elementos
Tem o foco no conteúdo.

```css
.botao::before {}
.cartao::after {}
.cabecalho::first-letter {}
.conteudo::first-line
```
